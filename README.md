# 지하철 노선도 미션
스프링 과정 실습을 위한 지하철 노선도 애플리케이션

# 요구사항 정리
- 지하철 역
  - 역 관리 API 기능 완성한다.
  - 지하철역 생성 시 이미 등록된 이름으로 요청한다면 에러를 응답한다.
- 지하철 노선
  - 같은 이름의 노선은 생성 불가하다.
  - 노선 관리 API 구현
    - [API 문서](https://techcourse-storage.s3.ap-northeast-2.amazonaws.com/d5c93e187919493da3280be44de0f17f#Line)
- 노선 기능에 대한 End to End 테스트 작성한다.
